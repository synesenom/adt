The goal

In this tutorial we create a simple application that displays a sequence of latitude/longitude coordinates on a map. To do so, we will use the Map widget in du.widgets, and add static and dynamic layers to it for the visualization. Once the map is built, the rest of the tutorial is just simple d3 tweaking to add interaction to the input forms.
