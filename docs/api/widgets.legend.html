du.widgets.legend

Module implementing a colored legend. A legend is a static label together with a colored square.

Requires: d3@v4 du.widgets.Widget

Legend

du.widgets.legend.Legend(name[, parent])

The legend widget class.
argtypedescription
namestringIdentifier of the legend.
parentobjectParent element to append widget to. If not specified, widget is appended to body. optional

text

du.widgets.legend.Legend.text(label)

Sets legend text.
argtypedescription
labelstringText of the legend.

color

du.widgets.legend.Legend.color(color)

Sets legend color. Default is white.
argtypedescription
colorstringColor to use.

highlight

du.widgets.legend.Legend.highlight(on)

Highlights legend. A highlighted legend has a bold text.
argtypedescription
onbooleanWhether to turn highlight on or off.
