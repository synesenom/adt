du.widgets.areachart

Module implementing an area chart. Part of the Analytics Dashboard Tools.

Requires: d3@v4 du.widgets.Widget

AreaChart

du.widgets.areachart.AreaChart(name[, parent])

The area chart widget class.
argtypedescription
namestringIdentifier of the widget.
parentobjectParent element to append widget to. If not specified, widget is appended to body. optional

xType

du.widgets.areachart.AreaChart.xType(type)

Sets the type of the X axis. Supported values are: number, time, string. Default is number.
argtypedescription
typestringType of the X axis.

opacity

Sets the opacity of the area plots. Default is 0.3.
argtypedescription
valuenumberThe opacity value to set.

data

du.widgets.areachart.AreaChart.data(data, scale)

Binds data to the area plot.
argtypedescription
dataArrayArray of x, y objects where X is a number or Date, Y is an object containing the y values for each area to plot.
scalenumberOptional scaling parameter. Each data point is divided by this value.
